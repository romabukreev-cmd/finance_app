# MVP Scope (Финансы)

## Входит в MVP
- Веб-приложение (Next.js) с адаптивным интерфейсом для ПК и мобильного браузера.
- Язык интерфейса: русский.
- Валюта: только RUB.
- Авторизация по почте (Supabase Auth), один пользователь на старте.
- Ручное добавление операций.
- Сущности: `accounts`, `categories`, `transactions`.
- Типы счетов: `asset`, `debt`.
- Типы операций: `income`, `expense`, `transfer`.
- Перевод (`transfer`) хранится как две связанные записи (`out` + `in`) с общим `transfer_id`.
- Балансы счетов считаются автоматически.
- Дашборд: капитализация, итоги месяца, карточки счетов, заготовка графиков.
- Страница операций: список и фильтры по месяцу/типу/счету/категории.
- Страница настроек: управление счетами и категориями.

## Не входит в MVP
- Мультивалюта.
- Импорт из банков/брокеров.
- Автогенерация дивидендов/купонов.
- Мультиюзерные сценарии и подписки.
- Сложная аналитика (годовые сравнения, прогнозы, KPI-модули).

## Неподвижные правила
- Пользователь вводит сумму как положительное число; знак задается типом операции.
- `income` участвует только в доходах.
- `expense` участвует только в расходах.
- `transfer` не участвует ни в доходах, ни в расходах, только в балансах счетов.
- Долг (`debt`) уменьшает общую капитализацию.
- Категория «Прочее» существует всегда и не удаляется.
- При удалении обычной категории связанные операции переводятся в «Прочее».

## Критерий готовности MVP
- Можно зайти по логину с ПК и телефона.
- Можно добавлять/редактировать/удалять счета и категории.
- Можно добавлять/редактировать/удалять доходы, расходы и переводы.
- Балансы, капитализация и помесячные итоги считаются корректно.
- Интерфейс пригоден для ежедневного ручного учета.
