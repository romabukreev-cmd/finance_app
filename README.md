# Финансы MVP

Личное веб-приложение учета финансов (ПК + мобильный браузер).

Текущий режим: **local-first** (данные хранятся в `localStorage`, без авторизации и без онлайн-базы).

## Стек
- Next.js + TypeScript
- Tailwind CSS + shadcn/ui

## Что уже работает
- Дашборд с расчетами:
  - капитализация,
  - доход/расход/разница за выбранный месяц,
  - карточки счетов,
  - последние операции,
  - расходы по категориям.
- Полный локальный CRUD счетов:
  - добавить,
  - редактировать,
  - архивировать,
  - удалить (если нет операций).
- Полный локальный CRUD категорий:
  - добавить,
  - редактировать,
  - архивировать,
  - удалить с автоматическим переносом операций в «Прочее».
- Полный локальный CRUD операций:
  - `income`,
  - `expense`,
  - `transfer` (как две ноги с общим `transferId`),
  - фильтры по месяцу/типу/счету/категории.
- Резервное копирование:
  - экспорт всех данных в JSON,
  - импорт из JSON с заменой текущих данных,
  - полный сброс хранилища,
  - авто-бэкап при первом открытии дня (хранится 3 последних копии).

## Быстрый запуск
1. Установи зависимости:
```bash
npm install
```
2. Запусти dev-сервер:
```bash
npm run dev
```
3. Открой [http://localhost:3000](http://localhost:3000)

## Технические заметки
- Линтер: `npm run lint`.
- Проверка типов: `npx tsc --noEmit`.
- В этой среде `next build` периодически падает на Windows `EPERM` (блокировка build-артефактов), это инфраструктурная проблема окружения.
